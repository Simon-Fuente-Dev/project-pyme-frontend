# Pocket Pyme - Frontend 🚀

<p align="center">
  <b>Pocket Pyme</b> es una plataforma web diseñada para digitalizar y potenciar el crecimiento de pequeñas y medianas empresas.
</p>

---

### 📋 Características Principales

<table>
  <tr>
    <td><b>Registro de Empresas</b></td>
    <td>Flujo de onboarding para que nuevas pymes se integren a la plataforma.</td>
  </tr>
  <tr>
    <td><b>Catálogo de Productos</b></td>
    <td>Panel administrativo para agregar, editar y listar la oferta comercial.</td>
  </tr>
  <tr>
    <td><b>Perfil de Contacto</b></td>
    <td>Herramientas de edición para actualizar datos de contacto y captar leads.</td>
  </tr>
  <tr>
    <td><b>Dashboard Moderno</b></td>
    <td>Interfaz limpia y funcional basada en estándares profesionales.</td>
  </tr>
</table>

### 🛠️ Stack Tecnológico

| Componente | Tecnología |
| :--- | :--- |
| **Core** | React (Functional Components & Hooks) |
| **Lenguaje** | TypeScript |
| **Interfaz (UI)** | Material UI (MUI) |
| **Estado** | Context API / Redux |
| **API** | Axios / Fetch |
